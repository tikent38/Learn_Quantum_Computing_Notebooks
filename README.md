# CAHSI-research
2026 Computing Alliance of Hispanic-Serving Institutions (CAHSI) research on Bridging Scientific Computing and Quantum Algorithms: An Educational Toolkit for Undergraduates  

This repository contains the notebooks developed as part of a CAHSI research project aimed at teaching quantum computing to students through the lens of comparing quantum computing ideas to classical computing ideas. The main files from this project can be found under the Jupyter_Notebook directory or by following the links below.
