# CAHSI-research
2026 Computing Alliance of Hispanic-Serving Institutions (CAHSI) research on Bridging Scientific Computing and Quantum Algorithms: An Educational Toolkit for Undergraduates  

This repo contains the notebooks developed as part of a CAHSI research project aimed at teaching quantum computing to students through the lens of comparing quantum computing ideas to classical computing ideas.
